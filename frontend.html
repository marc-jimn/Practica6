¿Qué es el Front–End?


El frontend es la parte de una aplicación web o sitio web con la que los usuarios interactúan directamente. En otras palabras, es la interfaz de usuario. El objetivo del frontend es crear una experiencia visual y funcional que sea intuitiva y atractiva para los usuarios.

Fundamentos:


  • - HTML (Lenguaje de Marcado de Hipertexto): Se utiliza para estructurar el contenido en la web. Define elementos como encabezados, párrafos, listas, enlaces imágenes, etc.
  • - CSS (Hojas de Estilo en Cascada): Se encarga de la presentación y el diseño del contenido HTML. Permite controlar el aspecto visual de la página, como colores, fuentes, márgenes y disposición de los elementos
  • - JavaScript: Añade interactividad y dinámicas a la página. Permite realizar acciones como validar formularios, manejar eventos, (como clics y desplazamientos), y crear animaciones.
